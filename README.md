# Desktop HERO RPG
Most of this is vibe coded, with codex, and refrences from claude. This program was a final project meant to be made polished with the help of vibe coding.

## Plot
You are a new student at this school, and you find that the students have a secert network of knowledge cards, and cheat sheets to help them make it through the classes. You hear about this junior call "The Val" and he's been sending out people to collect these cards. You're investigating to find out what's going on.

In this game, knowledge cards are essentially the attacks, and cheat sheets are the power ups. You have to use the box of knowledge cards you found to make it through varies tasks, and challenges.

## Highlights
* Infinite map genration, using randomized sprites of lockers, windows, and doors.
* Basic player movements simlar to 2D platformers (a-left, w-jump, d-right)
* Fight sequence simalar to Pokemon with a type system, and weaknesses
* Inventory, and a gaching system to obtain new attacks
